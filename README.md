# AI Hand Conductor

A hand gesture-controlled music conductor built using MediaPipe, OpenCV, and Pygame MIDI.

## Features

- Hand Tracking
- Volume Control
- Pitch Control
- Tempo Control
- Chord Mode
- 4 Musical Instruments(Piano, Flute, Organ, Violin)

## Technologies Used

- Python
- Numpy
- OpenCV
- MediaPipe
- Pygame MIDI

## How to Run

```bash
pip install mediapipe opencv-python pygame numpy
python hand_tracking.py
```
