# AI Hand Conductor

A hand gesture controlled music conductor built using MediaPipe, OpenCV and Pygame MIDI.

## Features

- Hand Tracking
- Volume Control
- Pitch Control
- Tempo Control
- Chord Mode
- Scale Switching

## Technologies Used

- Python
- OpenCV
- MediaPipe
- Pygame MIDI

## How to Run

```bash
pip install mediapipe opencv-python pygame numpy
python hand_tracking.py
```
